# CompanySearchSberActivity

Активити для поиска информации о компании по ИНН с использованием API Сбера.

## Описание

Активити позволяет получать информацию о компании по её ИНН через API Сбера, включая данные о юридических рисках.

## Требования

- PHP 7.4 или выше
- Расширения cURL и JSON
- Доступ к API Сбера

## Установка

1. Скопируйте папку `companysearchsberactivity` в директорию `/local/activities/`
2. В файле `config.php` укажите ваш API ключ Сбера
3. Активити будет доступно в конструкторе бизнес-процессов Битрикс24

## Параметры

### Входные параметры:
- `Inn` (обязательно) - ИНН компании для поиска

### Выходные параметры:
- `CompanyInfo` - информация о компании из API Сбера
- `LegalRisks` - юридические риски компании
- `CompanyData` - полные данные компании

## Пример использования

```php
// Пример вызова активити
$activity = new CompanySearchSberActivity('CompanySearchSber');
$activity->setProperty('Inn', '7707083893');
$result = $activity->execute();
```

## API Сбера

Документация по API: https://developers.sber.ru/docs/ru/sber-api/specifications/traffic-lights/traffic-lights-post

## Структура ответа API

Ответ от API Сбера содержит следующие поля:
- `inn` - ИНН компании
- `kpp` - КПП компании
- `ogrn` - ОГРН компании
- `name` - Наименование компании
- `address` - Адрес компании
- `legalForm` - Организационно-правовая форма
- `status` - Статус компании
- `risks` - Юридические риски (массив)
  - `type` - Тип риска
  - `description` - Описание риска
  - `severity` - Степень риска
  - `date` - Дата риска
